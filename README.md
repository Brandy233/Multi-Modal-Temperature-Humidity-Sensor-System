# Multi-Modal-Temperature-Humidity-Sensor-System
1.1 目的 随着物联网技术的发展，对环境参数（如温湿度、光照等）进行智能、远程、集中的监测与调控需求日益增长，尤其在智能家居、智慧农业、仓储管理等领域。 传统单点、本地化的传感器系统功能受限。本项目旨在设计并实现一个基于 STM32 的多模态温湿度传感器监控系统，该系统不仅能实现前端环境数据的实时采集与本地控制，还能通过通信模块将数据上传至后台服务器进行存储与分析， 并由上位机软件进行远程监控与管理，从而构建一个更完整、更智能的环境监测方案。 1.2 阅读建议 本报告主要面向环境监测、嵌入式系统开发、物联网应用、软件工程等领域的工程师、科研人员、技术管理人员。建议读者首先理解系统的整体 架构和核心功能，然后根据兴趣深入阅读前端嵌入式设计、通信协议、后台服务 及上位机应用等具体模块的实现细节
